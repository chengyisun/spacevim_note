# 原生 Vim 操作

紀錄我自己不熟的 Vim 操作

* `d`: 剪下選取的文字
* `0`: 跳到行首第一個字元
* `^(6)`: 跳到第一個 Word
* `$(4)`: 跳到最後一個 Word
* `H`: 游標移動到目前頁面的頂部
* `M`: 游標移動到目前頁面的中央
* `L`: 游標移動到目前頁面的底部
* `zt`: 以目前游標行為主, 移動頁面讓游標位於畫面頂部
* `zz`: 以目前游標行為主, 移動頁面讓游標位於畫面中央
* `zb`: 以目前游標行為主, 移動頁面讓游標位於畫面底部
* `gd`: go to definition
* `b`: 移動到單字的第一個字母
* `e`: 移動到單字的最後一個字母
* `w`: 移動到下一個單字的第一個字母 (note: b/w 感覺有點想是一組的, 而 e 就是移動到每個字母的尾巴)
* `[num] G`: 移動到第 [num] 行
